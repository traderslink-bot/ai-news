Summary

Veea Inc. announced the open-source release of Lobster Trap, a security tool that monitors and enforces policies on AI agent interactions with language models. Lobster Trap is MIT-licensed and integrated into TerraFabric, Veea's edge governance platform, and will be packaged inside NativelyAI's Native.Builder via LabLab.ai to enable policy-enforced deployment of AI agents in enterprise environments. The tool detects prompt injection, credential exposure, data leakage, and exfiltration with inline scanning performed in under a millisecond and runs within TerraFabric deployments. TerraFabric is currently in early deployments.

Positives
  • Open-source release under MIT license
  • Integrates with TerraFabric for edge governance
  • Partnership with NativelyAI and LabLab.ai to accelerate enterprise adoption
  • Enables inline policy enforcement on AI-agent interactions
  • Low latency scanning (under a millisecond) with no meaningful delay
  • Supports OpenAI-compatible backends with minimal code changes
  • Supports on-premises data sovereignty via TerraFabric
Negatives
  • TerraFabric is currently in early deployments, which may limit immediate adoption
  • Lobster Trap operates within TerraFabric deployments, requiring TerraFabric infrastructure for use
